# Quick Sort 快排序

## 算法思想

1. 从数组中挑出一个元素，作为基准(pivot)
2. 重新排列数组，比基准值小的放在基准的左边，比基准值大的放在右边
3. 再分别对左右区间重复第二步，直到各区间只有一个元素

## 实现方法

递归

## 复杂度分析

### 假设

* 数组长度为n

### 时间复杂度

**最优时间复杂度**：$O(n\log n)$

**最差时间复杂度**：O(n<sup>2</sup>)

**平均时间复杂度**：$O(n\log n)$

### 空间复杂度

**最优空间复杂度**：$O(\log n)$

**最差空间复杂度**：O(n)

**平均空间复杂度**：$O(\log n)$
