# Insertion Sort 插入排序

## 算法思想

每次将一个待排序的元素，按其大小插入到前面已经排好序的子序列中，直到全部元素插入完成为止。

## 实现方法

双重for循环

## 复杂度分析

### 假设

* 数组长度为n

### 时间复杂度

外循环遍历数组需要n时间，内循环遍历数组需要n时间

**最优时间复杂度**：数组已有序 O(n)

**最差时间复杂度**：数组逆序 O(n<sup>2</sup>)

**平均时间复杂度**：O(n<sup>2</sup>)

### 空间复杂度

在原数组上操作，空间复杂度为O(1)

## 稳定性分析

插入排序是稳定的排序算法，重复元素在排序前后的相对位置保持不变

## 扩展/优化

* 折半插入
